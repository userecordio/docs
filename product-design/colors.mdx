## Semantic Tokens

Semantic tokens are bound to the active theme via `[data-theme='Light']` or `[data-theme='Dark']`. Use `--base-*` tokens in components — they automatically adapt when the theme switches.

```css
/* Components consume base-* tokens */
.my-button {
  background-color: var(--base-primary);
  color: var(--base-primary-foreground);
  border: 1px solid var(--base-border);
}
```

<Info>
  Apply `data-theme="Light"` or `data-theme="Dark"` to the root `<html>` element to activate the correct token set.
</Info>

## Alpha Tokens

Alpha tokens are theme-aware overlays. Light theme uses white-based, dark theme uses zinc-950-based.

| Token | Light | Dark |

| `--alpha-10` | `rgba(255,255,255, 0.90)` | `rgba(9,9,11, 0.90)` |
| `--alpha-20` | `rgba(255,255,255, 0.80)` | `rgba(9,9,11, 0.80)` |

## Tailwind Palette Reference

Raw Tailwind color ramps are available as base tokens. Each step follows `--tailwind-colors-{ramp}-{step}`.

```css
/* Example: zinc ramp */
--tailwind-colors-zinc-50:  #fafafa;
--tailwind-colors-zinc-100: #f4f4f5;
--tailwind-colors-zinc-200: #e4e4e7;
--tailwind-colors-zinc-300: #d4d4d8;
--tailwind-colors-zinc-400: #a1a1aa;
--tailwind-colors-zinc-500: #71717a;
--tailwind-colors-zinc-600: #52525b;
--tailwind-colors-zinc-700: #3f3f46;
--tailwind-colors-zinc-800: #27272a;
--tailwind-colors-zinc-900: #18181b;
--tailwind-colors-zinc-950: #09090b;
```

Available ramps: `slate` · `gray` · `zinc` · `neutral` · `stone` · `red` · `orange` · `amber` · `yellow` · `lime` · `green` · `emerald` · `teal` · `cyan` · `sky` · `blue` · `indigo` · `violet` · `purple` · `fuchsia` · `pink` · `rose`

Steps available: `50 · 100 · 200 · 300 · 400 · 500 · 600 · 700 · 800 · 900 · 950`
