## Overview

Colors are organized into three layers: a **Tailwind base palette**, a set of **brand / custom colors**, and **semantic tokens** (`--base-*`) that map to either depending on the active theme (`Light` or `Dark`).

Always build UI with the semantic `--base-*` tokens below â€” they automatically adapt when `data-theme` switches between `Light` and `Dark`.

## Usage

<CodeGroup>

```css CSS
.button-primary {
  background-color: var(--base-primary);
  color: var(--base-primary-foreground);
  border: 1px solid var(--base-border);
}
```

```tsx Tailwind (React)
<button className="bg-primary text-primary-foreground border border-border">
  Save changes
</button>
```

```tsx Inline style (fallback)
<div style={{ backgroundColor: "var(--base-card)", color: "var(--base-card-foreground)" }}>
  Card content
</div>
```

</CodeGroup>
