## **Architecture**

Tokens are layered in three tiers: **base** (raw scales like spacing & color ramps), **aliases** (semantic intent like `--colors-primary-light`), and **themes** that bind aliases to the active mode via `[data-theme]`.

```text
/* base */
--spacing-4: 16px;
--tailwind-colors-zinc-900: #18181b;

/* alias */
--colors-primary-light: var(--tailwind-colors-zinc-900);

/* theme */
[data-theme='Light'] {
  --base-primary: var(--colors-primary-light);
}
```
