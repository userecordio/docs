## Scale

All spacing tokens follow `--spacing-{n}` where `n` maps to Tailwind's default spacing scale. The base unit is **4px**.


## Width & Height

Matching width (`--width-w-{n}`) and height (`--height-h-{n}`) tokens follow the same scale and are useful for setting explicit element dimensions.

```css
width: var(--width-w-10);    /* 40px */
height: var(--height-h-12);  /* 48px */
```

## Common Patterns

<CodeGroup>

```css Inline padding
/* Compact — xs buttons, tags */
padding: var(--spacing-1) var(--spacing-2);    /* 4px 8px */

/* Default — base buttons, inputs */
padding: var(--spacing-2) var(--spacing-4);    /* 8px 16px */

/* Relaxed — large buttons */
padding: var(--spacing-3) var(--spacing-6);    /* 12px 24px */
```

```css Section spacing
/* Card body */
padding: var(--spacing-6);    /* 24px */

/* Section vertical rhythm */
margin-top: var(--spacing-12);    /* 48px */
gap: var(--spacing-8);            /* 32px */
```

</CodeGroup>
