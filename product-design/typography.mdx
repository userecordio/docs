## Heading Components

| Element | Font Size | Line Height | Weight | Letter Spacing |
|---------|-----------|-------------|--------|----------------|
| H1 | `36px` / `48px` lg | `40px` | `800` (extrabold) | `-0.4px` |
| H2 | `30px` | `36px` | `600` (semibold) | `-0.4px` |
| H3 | `24px` | `32px` | `600` (semibold) | `-0.4px` |
| H4 | `20px` | `28px` | `600` (semibold) | `-0.4px` |
| p | `16px` | `28px` | `400` (normal) | `0` |
| lead | `20px` | `28px` | `400` (normal) | `0` |
| large | `18px` | `28px` | `600` (semibold) | `0` |
| small | `14px` | `14px` | `500` (medium) | — |

```css

font-family: var(--typography-typography-components-h1-font-family);
font-size: var(--typography-typography-components-h1-font-size);
font-weight: var(--typography-typography-components-h1-font-weight);
line-height: var(--typography-typography-components-h1-line-height);
letter-spacing: var(--typography-typography-components-h1-letter-spacing);

/* Body paragraph */
font-family: var(--typography-typography-components-p-font-family);

font-weight: var(--typography-typography-components-p-font-weight);
```

## Text Style Classes

The CSS file ships pre-composed utility classes following the pattern:

```
.text-{size}-leading-{normal|none}-{weight}
```

Examples:

```css
/* 14px / 20px line-height / semibold */
.text-small-leading-normal-semibold { ... }

/* 16px / 100% line-height / bold */
.text-base-leading-none-bold { ... }

/* 24px / 32px line-height / medium */
.text-2x-large-leading-normal-medium { ... }
```

Supported sizes: `extra-small · small · base · large · extra-large · 2x-large · 3x-large · 4x-large`

Supported weights per size: `regular · medium · semibold · bold · italic · underlined · strikethrough`
